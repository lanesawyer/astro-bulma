The `Notification` component renders Bulma's [notification element](https://bulma.io/documentation/elements/notification/) — a bold block used to display notices or alerts.

## Props

| Prop          | Type                                                          | Default | Description                            |
|---------------|---------------------------------------------------------------|---------|----------------------------------------|
| `color`       | `"link"` \| `"primary"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` | — | Bulma color modifier. |
| `light`       | `boolean`                                                     | —       | Light variant.                         |
| `dismissible` | `boolean`                                                     | —       | Adds a × delete button that removes the element. |
| `class`       | `string`                                                      | —       | Extra CSS classes.                     |
