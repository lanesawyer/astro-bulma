The `Title` and `Subtitle` components render Bulma's [title elements](https://bulma.io/documentation/elements/title/). Both accept a size from `1` (largest) to `6` (smallest) and render as `h1`/`h2` by default — use `as` to change the tag independently of the visual size.

## Basic example

<Preview>
  <Title>Title</Title>
  <Subtitle>Subtitle</Subtitle>
</Preview>

```astro
<Title>Title</Title>
<Subtitle>Subtitle</Subtitle>
```

## Sizes

<Preview>
  <Title size={1}>Title 1</Title>
  <Title size={3}>Title 3</Title>
  <Title size={5}>Title 5</Title>
</Preview>

```astro
<Title size={1}>Title 1</Title>
<Title size={3}>Title 3</Title>
<Title size={5}>Title 5</Title>
```

## Paired title and subtitle

When a title directly precedes a subtitle, add `spaced` to the title to keep space between them. Bulma recommends a two-step size difference:

<Preview>
  <Title size={3} spaced>Title 3</Title>
  <Subtitle size={5}>Subtitle 5</Subtitle>
</Preview>

```astro
<Title size={3} spaced>Title 3</Title>
<Subtitle size={5}>Subtitle 5</Subtitle>
```

## Custom tag

<Preview>
  <Title as="p" size={4}>A paragraph styled as a title</Title>
</Preview>

```astro
<Title as="p" size={4}>A paragraph styled as a title</Title>
```

## Props

Both components share the same props (except `spaced`, which is `Title` only):

| Prop     | Type                             | Default                            | Description                                       |
|----------|----------------------------------|------------------------------------|---------------------------------------------------|
| `size`   | `1 \| 2 \| 3 \| 4 \| 5 \| 6`     | —                                  | Size modifier, 1 (largest) to 6 (smallest).       |
| `as`     | `"h1"…"h6" \| "p"`               | `"h1"` (Title), `"h2"` (Subtitle)  | HTML tag to render.                               |
| `spaced` | `boolean`                        | —                                  | **Title only.** Keeps space before the subtitle.  |
| `class`  | `string`                         | —                                  | Extra CSS classes.                                |
| `...`    | `*`                              | —                                  | Any other HTML attributes.                        |
