The `Box` component renders Bulma's [box element](https://bulma.io/documentation/elements/box/) — a white padded container for grouping content.

## Basic example

<Preview>
  <Box>
    This content is inside a box.
  </Box>
</Preview>

```astro
<Box>
  This content is inside a box.
</Box>
```

## Props

| Prop    | Type     | Default | Description                         |
|---------|----------|---------|-------------------------------------|
| `class` | `string` | —       | Extra CSS classes on the `div.box`. |
| `id`    | `string` | —       | `id` attribute on the `div.box`.    |
| `...`   | `*`      | —       | Any other HTML `div` attributes.    |
