The `Progress` component renders Bulma's [progress bar element](https://bulma.io/documentation/elements/progress/) — a native `<progress>` for showing completion or loading state.

## Indeterminate

Omit `value` for an animated, indeterminate progress bar (useful for loading states).

<Preview>
  <Progress color="primary" />
</Preview>

```astro
<Progress color="primary" />
```

## Props

| Prop    | Type                                                                                         | Default | Description                                                  |
|---------|----------------------------------------------------------------------------------------------|---------|--------------------------------------------------------------|
| `color` | `"primary"` \| `"link"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"light"` \| `"dark"` | —   | Bulma color modifier.                                        |
| `size`  | `"small"` \| `"medium"` \| `"large"`                                                          | —       | Size modifier.                                               |
| `value` | `number`                                                                                      | —       | Current value. Omit for an indeterminate (animated) bar.     |
| `max`   | `number`                                                                                      | `100`   | Maximum value.                                               |
| `class` | `string`                                                                                      | —       | Extra CSS classes.                                           |
| `...`   | `*`                                                                                           | —       | Any other HTML progress attributes.                          |
