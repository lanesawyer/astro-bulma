The `Media` component renders Bulma's [media object](https://bulma.io/documentation/layout/media-object/). The default slot fills the content area; use the `left` slot for a figure (avatar, image) and the `right` slot for side actions. Nest `Media` components inside the default slot for threaded layouts like comments.

## Basic example

<Preview>
  <Media>
    <Image slot="left" src="https://bulma.io/assets/images/placeholders/128x128.png" size="64x64" alt="Avatar" />

        <br />
        Lorem ipsum dolor sit amet, consectetur adipiscing elit.
      </p>
    </div>
    <Delete slot="right" />
  </Media>

## Props

| Prop    | Type     | Default | Description                              |
|---------|----------|---------|------------------------------------------|
| `class` | `string` | —       | Extra classes on the `article.media`.    |
| `...`   | `*`      | —       | Any other HTML `article` attributes.     |

### Slots

| Slot      | Description                                |
|-----------|--------------------------------------------|
| (default) | Content of the `media-content`.            |
| `left`    | Content of the `media-left` figure.        |
| `right`   | Content of the `media-right` side.         |
