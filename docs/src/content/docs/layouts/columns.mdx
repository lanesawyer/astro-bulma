The `Columns` and `Column` components render Bulma's [column system](https://bulma.io/documentation/columns/). Add as many `Column` children as you want ‚Äî they'll share the width equally unless sized.

## Basic example

<Preview>
  <Columns>
    <Column><div class="notification is-primary">First</div></Column>
    <Column><div class="notification is-primary">Second</div></Column>
    <Column><div class="notification is-primary">Third</div></Column>
  </Columns>
</Preview>

```astro
<Columns>
  <Column>First</Column>
  <Column>Second</Column>
  <Column>Third</Column>
</Columns>
```

## Sizes and offsets

Size columns with a 12-column fraction (`1`‚Äì`12`) or a named fraction (`"half"`, `"one-third"`, ‚Ä¶):

<Preview>
  <Columns>
    <Column size={8}><div class="notification is-info">is-8</div></Column>
    <Column><div class="notification is-info">auto</div></Column>
  </Columns>
  <Columns>
    <Column size="half" offset="one-quarter">
      <div class="notification is-info">is-half is-offset-one-quarter</div>
    </Column>
  </Columns>
</Preview>

```astro
<Columns>
  <Column size={8}>is-8</Column>
  <Column>auto</Column>
</Columns>
<Columns>
  <Column size="half" offset="one-quarter">is-half is-offset-one-quarter</Column>
</Columns>
```

## Responsive sizes

For per-breakpoint behavior, pass Bulma's responsive classes directly:

```astro
<Column class="is-half-tablet is-one-third-desktop">‚Ä¶</Column>
```

## Columns props

| Prop        | Type          | Default | Description                                  |
|-------------|---------------|---------|----------------------------------------------|
| `multiline` | `boolean`     | ‚Äî       | Columns wrap onto multiple lines.            |
| `gapless`   | `boolean`     | ‚Äî       | Remove the gap between columns.              |
| `centered`  | `boolean`     | ‚Äî       | Center columns horizontally.                 |
| `vcentered` | `boolean`     | ‚Äî       | Align columns vertically.                    |
| `mobile`    | `boolean`     | ‚Äî       | Keep columns side by side on mobile.         |
| `desktop`   | `boolean`     | ‚Äî       | Only activate columns from desktop up.       |
| `gap`       | `0`‚Äì`8`       | ‚Äî       | Custom gap size (adds `is-variable`).        |
| `class`     | `string`      | ‚Äî       | Extra classes on the `div.columns`.          |
| `...`       | `*`           | ‚Äî       | Any other HTML `div` attributes.             |

## Column props

| Prop     | Type                          | Default | Description                                       |
|----------|-------------------------------|---------|---------------------------------------------------|
| `size`   | `1`‚Äì`12` or a named fraction  | ‚Äî       | Width (`"half"`, `"one-third"`, `"two-fifths"`, ‚Ä¶).|
| `offset` | `1`‚Äì`12` or a named fraction  | ‚Äî       | Left offset.                                      |
| `narrow` | `boolean`                     | ‚Äî       | Only take the width of the content.               |
| `class`  | `string`                      | ‚Äî       | Extra classes on the `div.column`.                |
| `...`    | `*`                           | ‚Äî       | Any other HTML `div` attributes.                  |
