The `Checkbox` component renders a Bulma [checkbox](https://bulma.io/documentation/form/checkbox/) inside a `<label>` element. Use the `label` prop for plain text, or the default slot for rich content.

## Basic checkbox

<Preview>
  <Checkbox name="agree" label="I agree to the terms" />
</Preview>

```astro
<Checkbox name="agree" label="I agree to the terms" />
```

## Rich slot content

Use the default slot instead of `label` to include HTML inside the label:

<Preview>
  <Checkbox name="rich">
    I agree to the <a href="#">terms of service</a> and <a href="#">privacy policy</a>
  </Checkbox>
</Preview>

```astro
<Checkbox name="rich">
  I agree to the <a href="#">terms of service</a> and <a href="#">privacy policy</a>
</Checkbox>
```

## Data attributes

`data-*` and `aria-*` attributes are forwarded to the `<label>` element; all other attributes are forwarded to the `<input>`:

<Preview>
  <Checkbox
    name="item"
    label="Item"
    data-item-id="42"
    aria-label="Select item 42"
  />
</Preview>

```astro
<Checkbox
  name="item"
  label="Item"
  data-item-id="42"
  aria-label="Select item 42"
/>
```
