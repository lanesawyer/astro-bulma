The `Textarea` component renders a Bulma [textarea](https://bulma.io/documentation/form/textarea/). Like `TextInput`, it wraps itself in `field`/`label`/`control` elements by default, or renders just the raw `<textarea>` with `standalone`.

## Basic example

<Preview>
  <Textarea name="message" label="Message" placeholder="Your message" rows={4} />
</Preview>

```astro
<Textarea name="message" label="Message" placeholder="Your message" rows={4} />
```

## Props

| Prop           | Type                                          | Default | Description                                            |
|----------------|-----------------------------------------------|---------|--------------------------------------------------------|
| `name`         | `string`                                      | —       | **Required.** `name` attribute.                        |
| `id`           | `string`                                      | `name`  | `id` attribute; defaults to `name`.                    |
| `label`        | `string`                                      | —       | Label text above the control.                          |
| `placeholder`  | `string`                                      | —       | Placeholder text.                                      |
| `value`        | `string`                                      | —       | Initial content.                                       |
| `rows`         | `number`                                      | —       | `rows` attribute.                                      |
| `required`     | `boolean`                                     | —       | Required field.                                        |
| `disabled`     | `boolean`                                     | —       | Disabled state.                                        |
| `readonly`     | `boolean`                                     | —       | Read-only state.                                       |
| `form`         | `string`                                      | —       | Associates the textarea with a form by id.             |
| `color`        | `"primary" \| "link" \| "info" \| "success" \| "warning" \| "danger"` | — | Color modifier.               |
| `size`         | `"small" \| "normal" \| "medium" \| "large"`  | —       | Size modifier.                                         |
| `fixedSize`    | `boolean`                                     | —       | Disables manual resizing (`has-fixed-size`).           |
| `help`         | `string`                                      | —       | Help text below the control.                           |
| `helpColor`    | same as `color`                               | —       | Color for the help text.                               |
| `standalone`   | `boolean`                                     | —       | Render only the `<textarea>`, no wrappers.             |
| `class`        | `string`                                      | —       | Extra classes on the `<textarea>`.                     |
| `controlClass` | `string`                                      | —       | Extra classes on the control div (e.g. `is-loading`).  |
| `...`          | `*`                                           | —       | Other native textarea attributes (maxlength, wrap, …). |
