The `Panel` component renders a Bulma [panel](https://bulma.io/documentation/components/panel/). Pass blocks via the default slot using Bulma's own classes (`panel-block`, `panel-icon`), and tab links via the `tabs` slot.

## Basic example

<Preview>
  <Panel heading="Repositories">
    <a href="#" class="panel-block is-active">astro-bulma</a>
    <a href="#" class="panel-block">bulma</a>
    <a href="#" class="panel-block">astro</a>
  </Panel>
</Preview>

## Props

| Prop      | Type                                                                   | Default | Description                        |
|-----------|------------------------------------------------------------------------|---------|------------------------------------|
| `heading` | `string`                                                               | —       | Text for the `panel-heading`.      |
| `color`   | `"primary" \| "link" \| "info" \| "success" \| "warning" \| "danger"` | —       | Color modifier.                    |
| `class`   | `string`                                                               | —       | Extra classes on the `nav.panel`.  |
| `...`     | `*`                                                                    | —       | Any other HTML `nav` attributes.   |

### Slots

| Slot      | Description                                    |
|-----------|------------------------------------------------|
| (default) | Panel blocks (`panel-block`, `panel-icon`, …). |
| `tabs`    | Links for the `panel-tabs` row.                |
