The `Dropdown` component renders a Bulma [dropdown](https://bulma.io/documentation/components/dropdown/) using the native `<details>`/`<summary>` elements — the browser handles the open/close toggle, so the component ships zero JavaScript. Pass menu items via the default slot using Bulma's own classes (`dropdown-item`, `dropdown-divider`).

Note that unlike a scripted dropdown, clicking outside or pressing <kbd>Escape</kbd> does not close an open menu — the user closes it by clicking the trigger again.

## Custom trigger

Style the trigger button with `triggerClass`, and replace its content with the `trigger` slot. Keep slot content non-interactive (spans, icons — not buttons or links), since nested interactive elements swallow the toggle click:

<Preview class="preview-tall">
  <Dropdown triggerClass="is-primary">
    <Fragment slot="trigger">
      <span>Custom trigger</span>
      <Icon icon="fa-angle-down" size="small" />
    </Fragment>
    <a href="#" class="dropdown-item">First item</a>
    <a href="#" class="dropdown-item">Second item</a>
  </Dropdown>
</Preview>

```astro
<Dropdown triggerClass="is-primary">
```

## Props

| Prop           | Type      | Default | Description                                             |
|----------------|-----------|---------|---------------------------------------------------------|
| `label`        | `string`  | —       | Text for the default trigger button.                    |
| `hoverable`    | `boolean` | —       | Also open on hover.                                     |
| `right`        | `boolean` | —       | Align the menu with the right edge of the trigger.      |
| `up`           | `boolean` | —       | Open the menu above the trigger.                        |
| `active`       | `boolean` | —       | Render initially open (`open` attribute).               |
| `triggerClass` | `string`  | —       | Extra classes on the trigger button (e.g. `is-primary`).|
| `class`        | `string`  | —       | Extra classes on the `details.dropdown`.                |
| `...`          | `*`       | —       | Any other HTML `details` attributes.                    |

### Slots

| Slot      | Description                                                    |
|-----------|----------------------------------------------------------------|
| (default) | Menu content (`dropdown-item`, `dropdown-divider`, …).         |
| `trigger` | Replaces the label/caret inside the trigger button.            |
