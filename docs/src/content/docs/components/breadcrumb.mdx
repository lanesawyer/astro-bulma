The `Breadcrumb` component renders Bulma's [breadcrumb navigation](https://bulma.io/documentation/components/breadcrumb/) from a flat array of items. The last item is automatically marked as active.

## Basic breadcrumb

<Preview>
  <Breadcrumb
    items={[
      { label: "Home", href: "/" },
      { label: "Ensembles", href: "/ensembles" },
      { label: "Chamber Orchestra" },
    ]}
  />
</Preview>

```astro
<Breadcrumb
  items={[
    { label: "Home", href: "/" },
    { label: "Ensembles", href: "/ensembles" },
    { label: "Chamber Orchestra" },
  ]}
/>
```

## Props

| Prop    | Type                                          | Default | Description                                                        |
|---------|-----------------------------------------------|---------|--------------------------------------------------------------------|
| `items` | `Array<{ label: string; href?: string }>`     | —       | **Required.** Array of breadcrumb items. Last item is always active.|
