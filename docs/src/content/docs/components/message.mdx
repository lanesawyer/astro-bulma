The `Message` component renders Bulma's [message element](https://bulma.io/documentation/components/message/) — a colored block for displaying informational text, warnings, or callouts. An optional `header` slot adds a title bar.

## Basic message

<Preview>
  <Message>
    This is a plain message body.
  </Message>
</Preview>

```astro
<Message>

</Message>
```

## Props

| Prop    | Type                                                               | Default | Description                              |
|---------|--------------------------------------------------------------------|---------|------------------------------------------|
| `color` | `"dark"` \| `"link"` \| `"primary"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` | — | Bulma color modifier. |
| `size`  | `"small"` \| `"medium"` \| `"large"`                               | —       | Size modifier.                           |
| `style` | `string`                                                           | —       | Inline styles on the outer `<article>`.  |
| `class` | `string`                                                           | —       | Extra CSS classes.                       |

## Slots

| Slot    | Description                                              |
|---------|----------------------------------------------------------|
| `header`| Rendered inside `<div class="message-header">`.         |
| default | Rendered inside `<div class="message-body">`.           |
