The `Menu` component renders a Bulma [menu](https://bulma.io/documentation/components/menu/) — a vertical navigation sidebar built from a `sections` array. Items support one level of nesting.

## Basic example

<Preview>
  <Menu
    sections={[
      {
        label: "General",
        items: [
          { label: "Dashboard", href: "#" },
          { label: "Customers", href: "#" },
        ],
      },
      {
        label: "Administration",
        items: [
          { label: "Team Settings", href: "#" },
          {
            label: "Manage Your Team",
            href: "#",
            active: true,
            children: [
              { label: "Members", href: "#" },
              { label: "Plugins", href: "#" },
            ],
          },
          { label: "Invitations", href: "#" },
        ],
      },
    ]}
  />
</Preview>

## Props

| Prop       | Type            | Default | Description                          |
|------------|-----------------|---------|--------------------------------------|
| `sections` | `MenuSection[]` | —       | **Required.** Menu structure.        |
| `class`    | `string`        | —       | Extra classes on the `aside.menu`.   |
| `...`      | `*`             | —       | Any other HTML `aside` attributes.   |

### Types

```ts
interface MenuSection {
  label?: string; // optional .menu-label heading
  items: MenuItem[];
}

interface MenuItem {
  label: string;
  href?: string;
  active?: boolean;
  children?: MenuItem[]; // one level of nesting
}
```
